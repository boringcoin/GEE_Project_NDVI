# NDVI 异常值检测方法对比结果

## 实验说明

本实验对论文 2.3.3 中提到的 3-Sigma、IQR、DBSCAN、LOF、LOF(预处理缺失点)、分位数法，以及当前流水线使用的 LOF+分位数组合方法进行比较。由于原始数据没有人工异常标签，本实验不使用监督分类准确率，而使用以下诊断指标：

- removed_rate：在原始有效观测中被判定为异常的比例。
- targeting_ratio：被剔除点相对季节基准的中位偏离 / 保留点相对季节基准的中位偏离，越大说明越集中剔除异常波动。
- jump_reduction_rate：过滤后相邻有效观测跳变幅度降低比例，越大说明时序更平滑。

## 结论

综合剔除比例、异常 targeting_ratio 和时序跳变降低效果，排名第一的方法为 **LOF**。因此，当前正式流水线已调整为仅使用 **LOF** 进行异常值过滤。该方法在剔除比例和局部异常识别能力之间取得较好平衡，可以减少分位数裁剪对真实物候峰值和谷值的过度删除风险。
