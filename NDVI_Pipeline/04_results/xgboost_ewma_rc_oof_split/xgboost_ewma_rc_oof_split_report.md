# EWMA-RC-XGBoost OOF残差校正实验

## 1. 划分口径

本实验严格采用测试期之前的全部历史数据进行最终预测建模：

- 训练/验证历史期：1985-01-01 至 2019-12-31
- 测试期：2020-01-01 至 2025-05-16

其中，最终基础模型使用 1985-2019 年全部数据训练。残差校正模块不直接使用测试标签，而是在 1985-2019 内部通过滚动 out-of-fold 方式生成残差训练样本。

## 4. 结果说明

`XGBoost_Lag3` 使用前三期NDVI作为输入，RMSE=0.0051，SMAPE=1.3468。加入EWMA趋势记忆后，`EWMA_XGBoost` 的RMSE=0.0045，SMAPE=1.2249。

最佳方案为 `EWMA_RC_XGBoost_OOF_s1`，RMSE=0.0034，SMAPE=1.4079。该结果用于判断残差校正模块在严格时间划分下是否真正提升最终预测效果。
