# XGBoost时间序列预测最终报告

## 1. 模型命名与实验口径

为避免实验名称过于技术化，本文将最终预测模型统一命名为三类：

- **XGBoost**：基础模型，输入为预测时刻之前3期NDVI。
- **EWMA-XGBoost**：在基础XGBoost上增加EWMA趋势记忆特征。
- **EWMA-RC-XGBoost**：在EWMA-XGBoost基础上增加残差校正模块。

需要强调的是，所谓“前三期NDVI”并不是只用三个数据点训练模型，而是指每一条训练样本的输入字段。模型训练仍然使用测试期之前的全部历史样本。

## 2. 数据划分

| 数据部分 | 时间范围 | 样本数 | 用途 |
| :--- | :--- | :--- | :--- |
| 训练/验证历史期 | 1985-01-01 ~ 2019-12-31 | 850000 | 训练最终基础模型；内部滚动OOF用于残差模块训练 |
| OOF残差验证折 | 2016, 2017, 2018, 2019逐年滚动 | 98000 | 生成训练期内部残差样本，避免测试集泄漏 |
| 测试集 | 2020-01-01 ~ 2025-05-16 | 131000 | 最终独立测试评价 |

最终基础模型使用1985-2019年全部历史样本训练，测试集为2020-2025年。残差校正模块不使用测试集标签，而是在训练/验证历史期内部通过滚动out-of-fold方式生成残差训练样本。

## 3. 模型结构

### 3.1 XGBoost

基础模型输入为：

`X_t = [NDVI_{t-1}, NDVI_{t-2}, NDVI_{t-3}]`

该模型用于检验仅依赖NDVI短期历史状态时的预测能力。

### 3.2 EWMA-XGBoost

EWMA-XGBoost在基础输入上增加一个EWMA趋势记忆项：

`X_t = [NDVI_{t-1}, NDVI_{t-2}, NDVI_{t-3}, EWMA_3(t-1)]`

EWMA用于将近期NDVI变化压缩为平滑历史状态。span=3时，平滑系数为 `alpha = 2 / (3 + 1) = 0.5`。该特征只使用预测时刻之前的信息，因此不存在未来信息泄漏。

### 3.3 EWMA-RC-XGBoost

EWMA-RC-XGBoost在EWMA-XGBoost之后增加残差校正模块。第一阶段先得到基础预测值：

`y_hat_base = f_XGB(X_t)`

第二阶段学习训练期内部OOF残差：

`r_t = y_t - y_hat_base`

最终预测为：

`y_hat_final = y_hat_base + y_hat_residual`

该模块的作用是修正主模型未解释的局部偏差。

## 4. 最终测试集结果

| 模型名称 | 模块组成 | 测试样本数 | MSE | MAE | RMSE | SMAPE | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| XGBoost | 基础XGBoost预测器 | 131000 | 0.00002553 | 0.0037 | 0.0051 | 1.3468 | 使用测试期之前全部历史样本训练，输入为前三期NDVI。 |
| EWMA-XGBoost | EWMA趋势记忆 + XGBoost | 131000 | 0.00001999 | 0.0031 | 0.0045 | 1.2249 | 在基础输入上增加EWMA短期趋势记忆特征。 |
| EWMA-RC-XGBoost | EWMA趋势记忆 + XGBoost + OOF残差校正 | 131000 | 0.00001182 | 0.0021 | 0.0034 | 1.4079 | 使用训练期内部OOF残差训练校正模块，不使用测试集标签。 |

从测试集结果看，EWMA-XGBoost相对于基础XGBoost同时降低RMSE和SMAPE，说明EWMA趋势记忆模块能够稳定提升预测效果。EWMA-RC-XGBoost进一步降低了MSE、MAE和RMSE，但SMAPE高于EWMA-XGBoost，说明残差校正模块更偏向降低绝对误差，对低NDVI区间的相对误差控制不足。

## 6. EWMA模块贡献

EWMA模块的作用可以概括为：用一个短期平滑记忆项表达近期NDVI变化趋势。相比基础XGBoost，EWMA-XGBoost的RMSE由 0.0051 降至 0.0045，SMAPE由 1.3468 降至 1.2249。因此，EWMA模块是本文最终预测模型中最稳定的有效改进。

## 7. 最终模型选择

若论文要求综合考虑MSE、MAE、RMSE和SMAPE，建议将 **EWMA-XGBoost** 作为最终主模型，因为它相对于基础XGBoost在绝对误差和相对误差上均有提升。

若论文更强调RMSE和MAE，可将 **EWMA-RC-XGBoost** 作为扩展模型或对照模型，因为它取得最低MSE、MAE和RMSE，但SMAPE略高。更稳妥的写法是：残差校正模块能显著增强绝对误差控制能力，但在相对误差指标上仍存在改进空间。

## 8. 论文建议表述

本文构建了融合EWMA趋势记忆的XGBoost NDVI时间序列预测模型。该模型在测试期之前全部历史样本上训练，并以预测时刻之前3期NDVI和EWMA短期记忆作为输入。实验结果表明，EWMA模块能够有效增强模型对NDVI短期连续变化的表达能力，使RMSE和SMAPE均优于基础XGBoost。进一步引入残差校正模块后，模型的MSE、MAE和RMSE继续下降，但SMAPE有所上升，说明残差校正更适合优化绝对误差。综合四项指标，本文最终采用EWMA-XGBoost作为主预测模型，并将EWMA-RC-XGBoost作为扩展对比模型。
