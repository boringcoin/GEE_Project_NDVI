## 关键口径确认

- 预测目标：基于预测时刻之前的历史NDVI信息，预测下一期15天窗口的重构NDVI。按监督学习样本展开时，每条样本的标签是当前15天窗口NDVI，输入只来自该窗口之前的历史窗口。
- 输入信息：每个样本只使用预测时刻之前的信息，包括 `NDVI_t-1`、`NDVI_t-2`、`NDVI_t-3` 和 `EWMA_3`。EWMA由历史NDVI递推得到，不包含当前或未来NDVI。
- 训练/测试：1985-2019为训练/验证历史期，2020-2025为最终测试期。
