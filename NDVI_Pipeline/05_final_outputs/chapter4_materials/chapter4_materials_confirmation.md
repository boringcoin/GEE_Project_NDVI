# 第四章材料确认说明

## 数据划分最终版

训练/验证历史期：1985-01-01—2019-12-31，850000个样本，用于训练模型。  
OOF残差验证折：2016—2019逐年滚动，98000个样本，用于残差模块训练。  
测试集：2020-01-01—2025-05-16，131000个样本，用于最终评价。

## 预测目标与信息泄漏说明

预测目标可以表述为：基于预测时刻之前的历史NDVI信息，预测下一期15天窗口的重构NDVI。按监督学习样本展开时，每条样本的标签是当前15天窗口NDVI，输入只来自该窗口之前的历史窗口。

每个样本输入只使用预测时刻之前的信息：`NDVI_t-1`、`NDVI_t-2`、`NDVI_t-3` 和由历史NDVI递推得到的 `EWMA_3`。不使用当前或未来NDVI，因此不存在未来信息泄漏。

## 表4.2使用建议

主文表4.2建议使用 `table4_2_basic_model_comparison.csv`。其中 XGBoost 和 LightGBM 为最新同口径结果；TimesFM、Informer、Autoformer、TFT、xLSTM为早期探索结果，需要在表注中说明不是最新同口径重跑。若评审要求严格同口径，主表只保留 XGBoost 与 LightGBM，并把其他早期模型放入补充说明。
